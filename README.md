## AX18固件特性：  
0.默认后台地址192.168.1.1；默认后台密码password。  
1.更新内容与https://github.com/sdf8057/ipq6000相同
2.完整的USB支持(大概) 
3.cpu超频至1.6ghz，跑分2w+。  
4.补齐部分扩展插件(可选)  
5.支持在线安装软件，手动安装请确保插件使用lua语言编写。  
6.此页面发布的AX18固件集成无线功能,2.4G无法正常显示传输速率
7.
